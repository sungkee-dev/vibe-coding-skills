# vibe-coding-skills

PRD 작성부터 화면 스펙 도출, 프론트엔드/백엔드 개발까지 이어지는 워크플로우 스킬 모음.

## 포함된 스킬

- **prd-generator** — PRD(제품 요구사항 문서)를 작성하여 `PRD.md`로 저장
- **prd-reviewer** — PRD를 개발자 관점에서 리뷰하고 보완
- **screen-spec** — PRD를 읽고 화면 목록과 구조 스펙을 `SCREEN_SPEC.md`로 출력
- **frontend-dev** — CDN React + Babel Standalone 방식의 프론트엔드 개발 가이드
- **backend-dev** — Supabase + Vercel 백엔드 연동 가이드

## 설치

처음 설치할 때는 마켓플레이스 등록을 먼저 해야 합니다.

```bash
# 1단계: 마켓플레이스 등록 (처음 한 번만)
claude plugins marketplace add github:sungkee-dev/vibe-coding-skills

# 2단계: 플러그인 설치
claude plugins install vibe-coding-skills
```

## 업데이트

```bash
claude plugins update vibe-coding-skills
```
